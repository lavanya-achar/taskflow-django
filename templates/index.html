Fix auth token refresh · TaskFlow Web Platform
Fix auth token refresh
CriticalIn ProgressBug

JWT refresh tokens are expiring incorrectly on mobile clients. The token refresh endpoint returns 401 even with a valid refresh token on iOS Safari. This affects all mobile users trying to maintain sessions beyond the initial login period.

# Steps to reproduce:
1. Login on iOS Safari
2. Wait for access token to expire (~15min)
3. Make any API call → receives 401
4. Refresh token endpoint also returns 401
